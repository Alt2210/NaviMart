# RESTful API Specification - AI Chef

Tài liệu mô tả chi tiết các endpoints được tự động đồng bộ từ mã nguồn backend.

## 1. getStatus
- **URL**: `/api/ai-chef/status`
- **Method**: `GET`
- **Description**: Whether the AI Chef integration is configured.
