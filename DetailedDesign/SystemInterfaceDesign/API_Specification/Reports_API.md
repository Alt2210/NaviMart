# RESTful API Specification - Reports

Tài liệu mô tả chi tiết các endpoints được tự động đồng bộ từ mã nguồn backend.

## 1. getDashboard
- **URL**: `/api/reports/dashboard`
- **Method**: `GET`
- **Description**: Family report dashboard.
- **Source File**: `reports.controller.ts`

---

## 2. getConsumptionTrends
- **URL**: `/api/reports/consumption-trends`
- **Method**: `GET`
- **Description**: Consumption and inventory event trends.
- **Source File**: `reports.controller.ts`

---

## 3. getWasteReport
- **URL**: `/api/reports/waste`
