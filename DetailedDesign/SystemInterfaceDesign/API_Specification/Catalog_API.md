# RESTful API Specification - Catalog

Tài liệu mô tả chi tiết các endpoints được tự động đồng bộ từ mã nguồn backend.

## 1. findAllCategories
- **URL**: `/api/catalog/categories`
- **Method**: `GET`
- **Description**: Active catalog categories.
- **Source File**: `catalog.controller.ts`

- **Source File**: `catalog.controller.ts`
