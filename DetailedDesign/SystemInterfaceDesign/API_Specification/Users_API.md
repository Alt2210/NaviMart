# RESTful API Specification - Users

Tài liệu mô tả chi tiết các endpoints được tự động đồng bộ từ mã nguồn backend.

## 1. getProfile
- **URL**: `/api/users/me`
- **Method**: `GET`
- **Description**: Current user profile.
- **Source File**: `users.controller.ts`

---

## 2. updateProfile
- **URL**: `/api/users/me`
- **Method**: `PATCH`
- **Description**: Current user profile updated.
- **Source File**: `users.controller.ts`
